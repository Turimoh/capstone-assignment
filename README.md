# Getting Started with TailwindCSS — Hello World

**Moringa School | GenAI Capstone | Week 1**

---

## 📌 About

A beginner-friendly Hello World project built with TailwindCSS. It renders a styled, interactive card component in the browser using only TailwindCSS utility classes — no custom CSS written.

---

## 🚀 How to Run

**No installation required.** This project uses the TailwindCSS CDN.

### Option 1 — Just open the file (simplest)

1. Download or clone this repository
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. That's it — the card will render immediately ✅

### Option 2 — Using VS Code Live Server

1. Open the project folder in [VS Code](https://code.visualstudio.com/)
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **"Open with Live Server"**
4. The page opens at `http://127.0.0.1:5500`

---

## 🎯 What You'll See

A centered white card on a purple-indigo gradient background containing:
- A "Capstone Project" badge
- A heading: **Hello, TailwindCSS! 👋**
- Three feature pills (Utility-First, No Custom CSS, CDN Setup)
- An interactive button — click it to see it turn green and show **🎉 It works!**
